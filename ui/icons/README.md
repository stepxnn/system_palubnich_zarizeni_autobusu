Složka `icons`

Sem vkládej ikony používané v rozhraní řidiče (levý sloupec).
Doporučené názvy souborů (odpovídají referencím v `panel-ridice.html`):
- nastaveni.png
- dispecink.png
- dalsi-zastavka.png

Doporučené formáty a rozměry:
- PNG nebo SVG (pro škálovatelnost preferuj SVG)
- doporučená velikost: 32x32 nebo 48x48 px
- průhledné pozadí

Pokud chceš, můžu vytvořit jednoduché placeholder ikony nebo převést SVG do PNG. Vlož sem své soubory a zkontroluji referenční cesty v `panel-ridice.html`.
